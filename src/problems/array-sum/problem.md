# 配列の合計を求める

数値の配列を受け取り、その**合計値**を返す関数 `sumArray` を実装してください。

## 仕様

- 関数名: `sumArray`
- 引数: `numbers` … 数値だけが入った配列(`number[]`)
- 戻り値: 配列内のすべての要素を足し合わせた数値

## 例

```js
sumArray([1, 2, 3]); // => 6
sumArray([10, -4, 2]); // => 8
sumArray([]); // => 0  (空配列の合計は 0)
```

## ヒント

- `for` ループや `Array.prototype.reduce` を使うと書けます。
- 空配列のときに `0` を返すことを忘れないでください。

`console.log()` でデバッグ出力できます。出力は実行結果のパネルに表示されます。
