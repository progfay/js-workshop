# 配列の応用

`filter` と `map` を組み合わせると、絞り込みと変換を続けて行えます。

```js
[...Array(5).keys()]; // -> [0, 1, 2, 3, 4]
(11).toString(16); // -> "b"  (16 進数の文字列)
```

## やってみよう

数値 `limit` を受け取り、`0` 以上 `limit` 未満の整数のうち**素数**だけを取り出し、
それぞれを **16 進数の文字列**に変換した配列を返す関数 `primeHexes` を実装してください。

```js
primeHexes(10); // -> ["2", "3", "5", "7"]
primeHexes(20); // -> ["2", "3", "5", "7", "b", "d", "11", "13"]
```

## ヒント

- 素数は「2 以上で、自分自身と 1 以外で割り切れない数」です。
- 16 進数への変換は `number.toString(16)` です。
