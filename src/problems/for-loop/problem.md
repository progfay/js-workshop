# forループ

`for` ループで同じ処理を繰り返せます。

```js
for (let i = 0; i < 10; i++) {
  console.log(i); // 0 から 9 まで
}
```

- `let i = 0` … 最初に一度だけ実行
- `i < 10` … 毎回チェックし、真の間ループを続ける
- `i++` … 毎回ループの最後に実行

## やってみよう

1. 変数 `limit` に数値 `10` を代入してください。
2. 変数 `total` を `0` から始め、`for` ループで `i` を `0` から `limit` 未満まで回し、
   各 `i` を `total` に足し込んでください（`total += i`）。

`console.log(total)` でデバッグ出力できます。
