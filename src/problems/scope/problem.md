# スコープ

「スコープ」は参照できる変数の集合です。関数の内側で宣言した変数は外からは見えません。
ネストした関数からは、親関数のスコープの変数を参照できます。
同じ名前の変数を内側で宣言すると、内側の宣言が優先されます（シャドーイング）。

## やってみよう

下のネストした関数の構造は**変更しない**でください
（`const` 宣言や関数の入れ子はそのまま）。

`secondFunction` の中に **1 行だけ**追加して、変数 `result` に
`"a: 1, b: 8, c: 6"` を代入してください。各スコープから正しい `a` / `b` / `c` を参照する必要があります。

```js
result = `a: ${a}, b: ${b}, c: ${c}`;
```

この行を**どの関数の中**に置くかで、参照される値が変わります。
