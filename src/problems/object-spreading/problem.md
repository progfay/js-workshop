# スプレッド構文

`...` を使うとオブジェクトを展開して新しいオブジェクトを作れます。
同じキーがある場合は**後に書いた方**で上書きされます。

```js
const a = { x: 1, y: 2 };
const b = { y: 3, z: 4 };
{ ...a, ...b }; // -> { x: 1, y: 3, z: 4 }
```

## やってみよう

2つのオブジェクト `a`, `b` を受け取り、`a` を `b` で上書きマージした新しいオブジェクトを返す関数 `merge` を実装してください。

```js
merge({ key1: "A", key2: "A" }, { key2: "B", key3: "C" });
// -> { key1: "A", key2: "B", key3: "C" }
```
