# 正規表現を使え、ルーク

正規表現で文字列からパターンに合う部分を抜き出せます。
`String.prototype.match` に `g` フラグ付きの正規表現を渡すと、一致した部分の配列が返ります（無ければ `null`）。

```js
"a1b2c3".match(/\d/g); // -> ["1", "2", "3"]
```

## やってみよう

文字列 `text` を受け取り、その中の**正しい郵便番号**だけを配列で返す関数 `extractZipcodes` を実装してください。

正しい郵便番号は次の2つの形式です。

- 数字 7 桁（例: `1234567`）
- 数字 3 桁 + `-` + 数字 4 桁（例: `111-2222`）

桁数が違うものや区切りが余分なものは含めません。返す各要素は前後の空白を取り除いて（`trim`）ください。

```js
extractZipcodes("郵便番号 1234567 です"); // -> ["1234567"]
```
