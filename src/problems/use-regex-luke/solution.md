# 解説

```js
function extractZipcodes(text) {
  const candidates = text.match(/[\d-]+/g) || [];
  return candidates.filter((s) => /^\d{7}$/.test(s) || /^\d{3}-\d{4}$/.test(s));
}
```

まず `/[\d-]+/g` で「数字とハイフンが続く部分」をまとめて取り出し、候補にします。
次に各候補を `^...$`（文字列全体が一致するか）でチェックし、`\d{7}` か `\d{3}-\d{4}` の
どちらかにぴったり一致するものだけを `filter` で残します。
`^` と `$` で全体に一致させるので、`12345678`（8 桁）や `123-4567-890`（区切りが余分）は
候補ごと弾かれます。`match` は一致が無いと `null` を返すため `|| []` で空配列にしています。
