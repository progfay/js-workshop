# 真偽値

真偽値 (boolean) には `true` と `false` の2つの値があります。

## Truthy / Falsy

偽値 (falsy) は `false`, `0`, `-0`, `0n`, `""`, `null`, `undefined`, `NaN` です。
それ以外はすべて真値 (truthy) で、`{}`, `[]`, `"0"` なども真値です。

## 比較演算子

`==` / `!=` は暗黙の型変換が起こります (`10 == "10"` は `true`)。
`===` / `!==` は型変換が起こりません (`10 === "10"` は `false`)。基本的に `===` / `!==` を使いましょう。

## `&&` と `||`

`&&` は左が偽値ならそれを、真値なら右を返します。
`||` は左が真値ならそれを、偽値なら右を返します。`true` / `false` を返すわけではありません。

```js
0 && 10; // -> 0
1 && 10; // -> 10
0 || 10; // -> 10
1 || 10; // -> 1
```

## やってみよう

各変数 `a1` 〜 `a12` を、コメントに書かれた式の**結果**に書き換えてください。
（文字列である必要はありません。式が返す値そのものを書きます。）

例えば `a1` のコメントは `0 == 1` なので、その結果である `false` を代入します。
