# 解説

```js
const a1 = false; // 0 == 1
const a2 = false; // 0 === 1
const a3 = true; // 10 == "10"  (型変換され等しい)
const a4 = false; // 10 === "10" (型が違うので等しくない)
const a5 = 0; // 0 && ""  (左が偽値 0 を返す)
const a6 = ""; // 1 && ""  (左が真値なので右の "" を返す)
const a7 = 0; // 0 && 10
const a8 = 10; // 1 && 10
const a9 = ""; // 0 || ""  (左が偽値なので右の "" を返す)
const a10 = 1; // 1 || ""  (左が真値 1 を返す)
const a11 = 10; // 0 || 10
const a12 = 1; // 1 || 10
```

`&&` / `||` は真偽値ではなく**オペランドそのもの**を返す点がポイントです。
