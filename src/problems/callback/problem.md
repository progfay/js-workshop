# コールバック関数

別の関数に渡す関数を**コールバック関数**と呼びます。
`setTimeout(callback, ms)` は、指定時間の後にコールバックを呼びます。
コールバックは**今すぐではなく後で**呼ばれる点がポイントです。

```js
setTimeout(() => {
  console.log("あとで呼ばれる");
}, 1000);
console.log("先に呼ばれる");
```

## やってみよう

コールバック `callback` を受け取り、`setTimeout` を使って**後で**それを呼ぶ関数 `delay` を実装してください。

```js
delay(() => {
  // setTimeout 経由で後から呼ばれる
});
```
