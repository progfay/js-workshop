# FizzBuzz

数値 `n` を受け取り、次のルールで結果を返す関数 `fizzbuzz` を実装してください。

- `3` でも `5` でも割り切れる → 文字列 `"FizzBuzz"`
- `3` で割り切れる → 文字列 `"Fizz"`
- `5` で割り切れる → 文字列 `"Buzz"`
- それ以外 → 数値 `n` をそのまま

```js
fizzbuzz(1); // -> 1
fizzbuzz(3); // -> "Fizz"
fizzbuzz(5); // -> "Buzz"
fizzbuzz(15); // -> "FizzBuzz"
```

## ヒント

- 割り切れるかは剰余演算子 `%` が `0` かどうかで判定します。
- `3` でも `5` でも割り切れる条件 (`% 15 === 0`) を**先に**判定するのがポイントです。
