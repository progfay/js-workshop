# ページネーション

ページ送り UI で表示するページ番号の並びを計算します。

`current`（現在ページ）, `total`（総ページ数）, `size`（表示するページ番号の個数）を受け取り、
現在ページが**できるだけ中央**に来るように、連続した `size` 個のページ番号の配列を返す関数 `getPageNums` を実装してください。

- ページ番号は `1` から `total` の範囲に収める
- 先頭・末尾付近では、はみ出さないように寄せる

```js
getPageNums(1, 5, 5); // -> [1, 2, 3, 4, 5]
getPageNums(3, 8, 6); // -> [1, 2, 3, 4, 5, 6]
getPageNums(8, 8, 3); // -> [6, 7, 8]
```

補助関数 `range(start, end)`（`[start, end)` の配列を返す）は用意済みです。**変更しないで**そのまま使ってください。
