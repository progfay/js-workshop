# オブジェクト

オブジェクトは、各要素を整数ではなく**キー**で識別する値の集まりです。

```js
const foodPreferences = {
  pizza: "yum",
  salad: "gross",
};
```

## やってみよう

変数 `pizza` を次のように定義してください。

```js
const pizza = {
  toppings: ["cheese", "sauce", "pepperoni"],
  crust: "deep dish",
  serves: 2,
};
```

`console.log(pizza)` でデバッグ出力できます。
