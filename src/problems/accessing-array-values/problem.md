# 配列の値にアクセスする

配列の要素には角括弧と添え字でアクセスします。添え字は `0` から始まります。

```js
const pets = ["cat", "dog", "rat"];
pets[0]; // -> "cat"  (最初の要素)
```

`pets.0` のようなドット記法ではアクセスできません。

## やってみよう

次の配列 `food` が与えられています。

```js
const food = ["apple", "pizza", "pear"];
```

`food` の **2 番目の要素**（添え字 `1`）を変数 `second` に代入してください。

`console.log(second)` でデバッグ出力できます。
