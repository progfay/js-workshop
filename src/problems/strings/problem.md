# 文字列

**文字列** は引用符でくくった値です。一重引用符と二重引用符のどちらも使えますが、
このアプリでは二重引用符に統一します。

```js
"this is a string";
```

## 文字列結合

`+` 演算子やテンプレートリテラルで結合できます。

```js
"名前: " + "progfay";
`価格: ${1000}円`;
```

## やってみよう

変数 `someString` を作り、文字列 `"this is a string"` を代入してください。

```js
const someString = "this is a string";
```

`console.log(someString)` でデバッグ出力できます。
